# Zomato-Data-science-Project
Data Science Project showcasing Exploratory Data analysis through Python and it's libraries

Zomato Data Analysis Project
Overview

This project focuses on analyzing Zomato restaurant data using Python to uncover meaningful insights about restaurant ratings, online ordering trends, table booking preferences, restaurant types, and customer behavior.

The project follows a structured data analysis workflow, including data loading, data cleaning, exploratory data analysis (EDA), and visualization to transform raw data into actionable insights.

Dataset
Zomato Restaurant Dataset

The dataset contains information about restaurants listed on Zomato, including:

Restaurant Name
Location
Online Order Availability
Table Booking Availability
Ratings
Votes
Approximate Cost for Two People
Restaurant Type
Cuisine Information

The dataset was analyzed to understand customer preferences and restaurant trends.

Project Workflow
1. Data Loading
Imported the Zomato dataset into Python.
Examined dataset structure and column information.
Checked data types and dataset dimensions.
2. Data Cleaning
Identified missing values.
Removed null values and unnecessary records.
Handled inconsistent data formats.
Prepared a clean dataset for analysis.
3. Exploratory Data Analysis (EDA)

Performed detailed analysis to understand:

Restaurant distribution across categories
Online ordering trends
Table booking preferences
Customer ratings and reviews
Cost distribution among restaurants
Popular restaurant types
4. Data Visualization

Created insightful visualizations using:

Count Plots
Bar Charts
Histograms
Pie Charts
Heatmaps
Distribution Plots

Visualizations were developed using Matplotlib and Seaborn to identify trends and patterns effectively.

Key Insights

Some important insights generated from the analysis include:

Customer ratings distribution across restaurants.
Impact of online ordering on restaurant popularity.
Relationship between restaurant cost and ratings.
Most common restaurant types and cuisines.
Customer preferences regarding table booking facilities.
Restaurant categories with higher customer engagement.
